# Blind Auction Program
딕셔너리를 활용한 간단한 프로그램 예제입니다.
---
# 실행순서
1. art.py에 저장된 로고를 출력하고, 입찰자의 이름을 입력받습니다.<br>
2. 해당 입찰자의 입찰가를 입력받습니다.<br>
3. 모든 출력을 초기화 합니다.<br>
4. 위의 2 ~ 3의 과정을 입찰 인원 수만큼 반복합니다.<br>
---
