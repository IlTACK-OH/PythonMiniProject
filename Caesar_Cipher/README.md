# Caesar Cipher
카이사르 암호 생성기이자 해독기입니다.
> encode Mode<br>
암호화할 문자열을 공백없이 입력한 후 shift number를 입력합니다.

> Decode Mode<br>
복호화할 문자열을 공백없이 입력한 후 shift number를 입력합니다.
